# `fonts/` 目录说明

`fonts/` 目录是可选的。

如果你希望在 `macOS` / `Linux` 本地编译时尽量接近 Windows 下的中文字体效果，可以把下面 4 个字体文件放到本目录：

- `simsun.ttc`：宋体
- `simhei.ttf`：黑体
- `simfang.ttf`：仿宋
- `simkai.ttf`：楷体

这些字体通常可以从 Windows 的 `C:\Windows\Fonts` 获取。

目录结构示例：

```text
fonts/
├── simsun.ttc
├── simhei.ttf
├── simfang.ttf
└── simkai.ttf
```

如果不需要这些字体，这个目录可以留空。

出于版权原因，不建议将这些字体提交到公开仓库。
